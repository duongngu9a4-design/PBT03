Giới Thiệu

Repository này được xây dựng để thực hiện Phiếu Bài Tập 3 của học phần phát triển Web. Nội dung bài tập tập trung vào các kiến thức nền tảng của CSS bao gồm Selectors, Box Model, Inheritance và Cascade.

Thông qua bài tập, sinh viên thực hành cách lựa chọn phần tử HTML bằng CSS, kiểm soát bố cục thông qua Box Model, hiểu cơ chế kế thừa thuộc tính và cách trình duyệt xác định quy tắc CSS được ưu tiên áp dụng.

---

Nội Dung Thực Hiện

1. CSS Selectors

- Element Selector
- Class Selector
- ID Selector
- Descendant Selector
- Child Selector
- Pseudo-class Selector

2. CSS Box Model

- Content
- Padding
- Border
- Margin
- Kích thước phần tử
- Khoảng cách giữa các thành phần giao diện

3. CSS Inheritance

- Các thuộc tính được kế thừa
- Các thuộc tính không được kế thừa
- Ảnh hưởng của phần tử cha đến phần tử con

4. CSS Cascade

- Độ ưu tiên của CSS
- Specificity
- Thứ tự khai báo
- Inline Style, Internal Style và External Style
